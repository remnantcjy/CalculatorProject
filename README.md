# CH2 계산기 과제
- **Spring 9기 Java - CH2 계산기 과제**  
자바 문법을 활용하여 사칙연산 계산기를 구현했습니다.

<br>

## 📌 프로젝트 소개
사용자로부터 값을 입력받아 사칙연산을 수행하고, 계산 결과를 출력하는 간단한 콘솔 기반 계산기 프로그램입니다.

<br>

## ⚙️ 주요 기능
- 사칙연산 (+, -, *, /) 수행 
- 0으로 나누는 연산 방지 
- 잘못된 입력 시 재입력 유도
- 연산 결과 목록 관리 (삭제, 비교 기능)
<br>

## 👩🏻‍💻 기술 스택
- **Language**: Java 17  
- **IDE**: IntelliJ IDEA  
<br>

## 🗓 개발 기간
- 2025.10.14 ~ 2025.10.21
